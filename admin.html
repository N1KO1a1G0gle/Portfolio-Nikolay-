This editor saves changes straight to your GitHub repo. You'll need a personal access token scoped to only this repository — it never leaves your browser.

  1. Go to github.com/settings/personal-access-tokens/new
  2. Under Repository access, choose Only select repositories → Portfolio-Nikolay-
  3. Under Permissions → Repository permissions, set Contents to Read and write
  4. Generate the token and paste it below
