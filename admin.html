Connect to your site

Two ways to edit — pick whichever fits where you are.

On this computer works offline

Edits save straight into the website folder on your disk — ideal alongside VS Code. Publish later with a git push (or ask Claude to push).

Folder editing needs Chrome or Edge, opened via a localhost address (e.g. VS Code Live Server) — not a file:// page.

The live site from anywhere

Changes commit to GitHub and go live in about a minute. Needs a personal access token scoped to only this repository — it never leaves your browser.

  1. Go to github.com/settings/personal-access-tokens/new
  2. Under Repository access, choose Only select repositories → Portfolio-Nikolay-
  3. Under Permissions → Repository permissions, set Contents to Read and write
  4. Generate the token and paste it below
